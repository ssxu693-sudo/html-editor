# HTML 编辑器

一个基于浏览器的本地 HTML 可视化编辑器，支持直接编辑文本、替换图片/视频、字体样式调整，并可导出修改后的文件。

## 功能

- **可视化编辑** - 在预览区直接点击文字编辑，点击图片/视频替换
- **字体调整** - 修改字号（缩放比例）和颜色（拾色板 + HEX 输入）
- **撤销/重做** - 支持 Ctrl+Z / Ctrl+Shift+Z
- **文件打开** - 支持直接打开单个 HTML 文件或打开文件夹（含资源目录）
- **源码编辑** - 高级模式查看和编辑 HTML 源码
- **导出** - 导出新文件（含资源自动复制），或覆盖原文件

## 使用方式

直接在浏览器中打开 `index.html`（推荐使用 Chrome、Edge 或 Arc）。

## 技术栈

- 纯原生 HTML/CSS/JS，无框架依赖
- File System Access API（浏览器原生）
- Google Fonts（JetBrains Mono + Noto Serif SC）

## 设计风格

Ink Press — 精密工具感设计，等宽字体主导，60px 网格背景，橙红色调。

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| Esc | 关闭源码面板 |
